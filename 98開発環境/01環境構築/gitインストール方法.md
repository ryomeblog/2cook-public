# Git Bashの実行

1. Windowsスタートメニューを開き、「Git」フォルダから「Git Bash」を選択し、コンソールを開きます。

![21](./img/git22.png)

2. 下のような画面が表示されていれば起動は問題ありません。

![22](./img/git23.png)

3. 「ls」コマンドを入力し、ディレクトリ内の一覧を表示してみましょう。表示に問題が無ければインストールは終了です。

![23](./img/git24.png)

4. お疲れさまでした。
